# Foodie Cart

This project cart implementation for online food delivery app using react, redux, bootstarp and material-ui

## Getting Started
- clone repo
- `npm install`
- `npm start`

## Built With
- React
- Redux
- Bootstarp
- Material Ui
- React Router
- Font Awesome
